# What is Jupyter Notebook?

Jupyter Notebook is an open-source web-based tool that allows you to create and share documents that contain live code, equations, visualizations, and explanatory text. It supports multiple programming languages (especially Python) and is widely used in data science, machine learning, and AI development.

It combines code execution with documentation, which makes it ideal for learning, experimentation, and collaboration.




### Two Use Cases for Jupyter Notebook in AI Development
##### Model Development & Experimentation
Jupyter Notebooks let AI developers build and test machine learning models step by step. You can load data, try different algorithms, tune hyperparameters, and visualize results — all in one document. This is great for fast experimentation and debugging.

##### Educational & Research Work
In education and research, Jupyter is used to teach AI concepts, demonstrate code, and explain algorithms with visual aids. Because you can mix code, math, and text in one place, it’s perfect for tutorials, research papers, and data science projects.
